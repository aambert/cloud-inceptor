Installation Instructions

The OpenVPN configuration does two factor authentication using both a user/password and client certificate. Once a VPN client is configured the same user and password used to access this page should be used to log in to the VPN.

The OpenVPN Connect client for Microsoft Windows and Apple OSX can be downloaded directly from the OpenVPN website. Once downloaded extract the openvpn-config.tunnelblick.zip archive and import the *.ovpn from within the unzipped folder. Alternate clients are made available for download above in case you are unable to download directly from the OpenVPN website. Other installation methods are outlined below.

Android

For Android devices you need to install the OpenVPN client application from the Google Appstore. Once installed download openvpn-config.client.zip file directly to the device by clicking on the link above. The archive should download to the device's local downloads folder. You can unzip the file from within the device directly into the same download location. Once the archive has been extracted open the OpenVPN application and import the *.ovpn file from within the extracted folder. You can save the user and password in the OpenVPN App by editing the imported profile, if you do not wish to reenter the credentials each time you connect.

Apple IOS

For Apple IOS devices you need to install the OpenVPN client application from the Apple Appstore. Once installed download the openvpn-config.mobileconfig profile directly to the device by clicking on the link above. Then follow the steps to install the profile, which will appear as a line item in your IOS settings App. Once the profile has been installed the VPN connection can be initiated from the OpenVPN App. You can save the user and password in the OpenVPN App by editing the imported profile, if you do not wish to reenter the credentials each time you connect.

Apple OSX

For Apple OSX MacBooks install the Tunnelblick VPN client. Once installed download the openvpn-config.tunnelblick.zip file and unzip it. Next doubleclick on the *.tblk artifact that is extracted from the archive to install the configuration to the Tunnelblick client.  

Microsoft Windows

For Microsoft Windows download and install the correct client for the version of Windows you are using from the list of three MSIs above. Once installed simply download and unzip the openvpn-config.tunnelblick.zip to the config folder within the VPN client's installation folder.
